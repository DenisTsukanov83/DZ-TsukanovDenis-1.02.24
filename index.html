Задача №1

Напишите регулярное выражение для поиска всех слов, начинающихся с определенной буквы.

Text:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Reiciendis architecto iste non qui sunt quos, aliquam neque. Ipsa, odit autem. 
    
    const string1 = 'Lorem ipsum dolor sit amet consectetur adipisicing elit. Harum, autem.';
    let a = prompt('Введите букву для поиска');
    let res1 = new RegExp(`\\b${a}\\w{1,}\\b`, 'gi');
    alert(string1.match(res1));

Задача №2

Создайте функцию, которая проверяет, является ли строка датой в формате "дд-мм-гг" с использованием регулярных выражений.

    
    const string2 = prompt('Введите строку');
    if(string2.match(/^(0?[1-9]|[12][0-9]|3[01])-(0[1-9]|1[0-2])-(\d\d)$/)) {
        alert('Эта строка дата в формате "дд-мм-гг"');
    } else {
        alert('Эта строка НЕ в формате "дд-мм-гг"');
    }

Задача №5

Создайте функцию, которая проверяет, является ли строка палиндромом с использованием регулярных выражений.

    
    const string = prompt('Введите слово'),
        string2 = string.replaceAll(' ', '')
    let regexp = /[a-zа-я]/iy, start, end, success = true;

    for(let i = 0; i < string2.length; i++) {
        regexp.lastIndex = i;
        start = regexp.exec(string2);
        regexp.lastIndex = string2.length - 1 - i;
        end = regexp.exec(string2)
        if(start[0].toLocaleUpperCase() != end[0].toLocaleUpperCase()) success = false;
    }
    success ? alert('Это полендром') : alert('Это НЕ полендром');
